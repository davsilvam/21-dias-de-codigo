# **Site com API Pública**

## :smirk_cat: **Projeto**

Este projeto faz parte do desafio **_21 Dias de Código_** e se baseia em uma pequena página que gera gifs aleatórios de gatos consumindo a _[TheCatAPI](https://docs.thecatapi.com)_.
