# **Site com API Pública**

## :smirk_cat: **Projeto**
Este projeto faz parte do desafio ***21 Dias de Código*** e se baseia em uma pequena página que gera gifs aleatórios de gatos consumindo a *[TheCatAPI](https://docs.thecatapi.com)*.
